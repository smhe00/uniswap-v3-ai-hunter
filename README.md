## 1. 核心方案架构
该系统采用 **“降频训练 + 高频执行”** 的混合架构：
- **大脑层 (Brain)**：基于 1 小时级别数据训练的 XGBoost + LSTM + GA 模型，过滤市场噪音，识别真实趋势。
- **神经层 (Execution)**：每 **15 分钟** 进行一次全量市场扫描，毫秒级响应潜在瀑布风险。
- **混合执行逻辑**：

    - **入场锁定**：部署后进入 2-4 天的稳定期（除非触发警报），以节省 Gas 费并最大化复利。

## 2. 非对称避险策略 (Asymmetric Logic)
当系统检测到风险决定撤出 LP 池时，会根据趋势判定持有不同资产：
- **强势看多 (EMA金叉 + RSI > 58)**：撤退并 **100% 持有 ETH**（吃满涨幅，避开 LP 卖出损耗）。
- **强势看空 (EMA死叉 + RSI < 42)**：撤退并 **100% 持有 USDC**（锁定美金价值，躲避瀑布）。
- **方向不明**：**原样持有 (Keep Ratio)**（不进行 Swap 兑换，节省手续费和滑点）。
